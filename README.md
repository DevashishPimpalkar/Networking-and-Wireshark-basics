## Why Wireshark?
Packet captures are rich data sources.
By analyzing real traffic, we can understand:
- Device intent (DNS)
- Trust relationships (ARP)
- Protocol behavior (TCP vs UDP)

This approach builds strong fundamentals for
**Cybersecurity**, **Network Defense**, and future **AI-based traffic analysis**.
